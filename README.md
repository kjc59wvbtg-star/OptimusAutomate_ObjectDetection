# OptimusAutomate_ObjectDetection
# Object Detection using YOLOv8

## Project Overview

This project implements an Object Detection application using the YOLOv8 model and Streamlit. Users can upload images, and the model automatically detects objects, draws bounding boxes, and displays confidence scores.

## Features

- Image upload support
- YOLOv8 object detection
- Bounding box visualization
- Object labels
- Confidence scores
- Streamlit web interface

## Technologies Used

- Python
- YOLOv8
- Ultralytics
- Streamlit
- OpenCV
- Pillow

## Results

The application successfully detects objects in uploaded images and displays detection summaries with confidence scores.
